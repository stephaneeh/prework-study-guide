HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The heading elements should be listed in numerical order
  • ul is an unordered list (or bullet points)
  • ol is an ordered list (includes numbers)
  • Images can be included using "< img src="LocationOfImage"/ >"
  • Hyperlinks can be including using "< a href="EnterURL">Hyperlink name"

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A special wild card symbol * can be used to apply a rule to all elements visiable on the page
  • DRY stands for Do Not Repeat Yourself - where possible, reuse pieces of code to reduce th number of lines
  • A class attirbute can be used to add styles to certain elements, not all.

Git

  • git status - what branch am i in (main/feature)
  • git checkout main - switch back to main branch
  • git checkout -b feature/name - create a feature branch called name
  • git add -A - prepare staging area (after changes made to code
  • git commit -m"what did i change" - commit these changes with a description of what i changed
  • git pull origin main - double check local branch is synced to base branch
  • git push origin feature/"name" - push changes to remote branch
  • Move over to GitHub and Compare & pull request. Update contents before creating pull request and merge

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is a set of instructions that tells the computer how to perform a task
  • An array is a single variable that is used to hold a group of data
  • " " indicates a string, 42 indicates a number, true indicates a boolean
  • console.log outputs a message to the web console
  • if checks if the first condition provided is true, else if checks the second condition
  • === checks if 2 values are equal, and returns a boolean result
  • x++ is shortcut for x+1 (although the same number of characters
  • /* */ comments out code
